# Freelancer Marketplace

A full-stack freelancer marketplace application built with Node.js, Express, MongoDB, and React.

## Project Structure

- **Backend**: Node.js, Express, MongoDB (Mongoose), TypeScript.
- **Frontend**: React, TypeScript.
